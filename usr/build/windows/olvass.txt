
Ebben a directoryban a program builder implicit szabályaihoz tartozó
batch filék vannak. Pl., ha BUILD úgy látja, hogy MSK-ból SAY filét
kell csinálni, akkor ezt az MSK2SAY.BAT filé végrehajtásával teszi. 
Ezeket a batch filéket BUILD a %BUILD_BAT%-ban megadott helyen keresi.


A BUILD program környezeti változókon keresztül kommunikál a batch-ekkel.
A batch-ek által használt környezeti változók az alábbiak:

BUILD_CLS  cls directory, objccc paramétere, pl. set BUILD_CLS=-iobjdef
BUILD_OBJ  object directory a különböző platformú binárisok szétválasztásához
BUILD_OPT  opciós filé CPP fordításhoz cccdir\usr-ben
BUILD_BAT  implicit szabályokhoz rendelt batchek directoryja
BUILD_INC  include directoryk (;)
BUILD_EXE  exe directory
BUILD_SYS  NT subsystem, MSC-vel windows/console
BUILD_LPT  standard library path 
BUILD_LIB  standard libraryk
BUILD_PRE  a prg2ppo preprocesszornak átdott paraméterek
BUILD_LEX  a flex-nek átadott kapcsolók
BUILD_BSN  a bison-nak átadott kapcsolók
BUILD_CPP
